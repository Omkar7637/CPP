# Topic 01 - Practice

## A) Write and run

1. Print your name, role, and target company in one formatted line.
2. Read two integers and print sum, difference, product, quotient, remainder.
3. Read radius and print area of circle using `double`.
4. Swap two numbers using third variable.
5. Convert Celsius to Fahrenheit.

## B) Interview mini-drills

- Explain why `scanf("%d", x);` is wrong.
- Explain integer division with 3 examples.
- Explain the output of:
  - `int x = 7; printf("%d", x / 2);`
  - `double y = 7 / 2; printf("%.2f", y);`
- Explain precedence in:
  - `10 + 2 * 3`
  - `(10 + 2) * 3`

## C) Submission checklist (for yourself)

- Code compiles without warnings
- You tested normal and edge input
- You can explain each line in your own words
