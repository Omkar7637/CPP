# Topic 03 - Practice

## Coding tasks
1. Reverse array in-place.
2. Find max/min in array.
3. Linear and binary search.
4. String length without `strlen`.
5. String reverse in-place.
6. Palindrome string checker.
7. Count vowels/consonants.
8. Pointer-based array traversal only.
9. Swap two numbers using pointers.
10. Implement `my_strcpy` and `my_strcmp`.

## Interview drills
- Explain array decay in function parameters.
- Explain `const char *p` vs `char * const p`.
- Explain why string literals should be treated read-only.
