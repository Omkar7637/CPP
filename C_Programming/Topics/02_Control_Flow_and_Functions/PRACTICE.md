# Topic 02 - Practice

## A) Coding tasks

1. Check even or odd using if-else.
2. Find greatest among 3 numbers.
3. Build simple grade calculator using else-if ladder.
4. Menu-based calculator using switch-case.
5. Sum of first N natural numbers using for loop.
6. Reverse a number using while loop.
7. Print multiplication table using do-while.
8. Create `max(a, b)` function and use it in program.
9. Recursive factorial with input validation.
10. Recursive Fibonacci for first N terms.

## B) Interview drills

- Explain fall-through in switch-case.
- Difference between while and do-while.
- Explain why C is pass-by-value.
- Explain how recursion uses stack.

## C) Quality checklist

- Compiles cleanly
- Handles invalid input
- Has meaningful function names
- No duplicate logic blocks
